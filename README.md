### Quick Start
1. Installing dependencies
```sh 
composer install
```
```sh
npm install
```
2. Migrate the database
```sh 
php artisan migrate
```
3. Run the application in development mode with hot reloads
```sh 
php artisan serve
```  
```sh 
npm run dev
```
